# create-files-agent — PPTX Format Skipped

**Date**: 2026-05-28
**Status**: Adopted
**Context**: PR-AR-6 punted docx/pdf/pptx/xlsx. v3-B3 fills three, but pptx has no MIT/BSD-licensed option; only unidoc/unioffice (AGPL) exist.

**Decision**: Do not implement pptx; `format=pptx` returns `tool.Error("pptx format not supported (no permissive-licensed library); please use docx instead")`.

**Reconsider when**: MIT/Apache-licensed pptx library appears; or user explicitly accepts AGPL.
