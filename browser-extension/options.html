The base URL of your Hermind server
Set this in your Hermind config.yaml under browser_extension.api_key

How to use

  1. Enter your Hermind server URL and API Key above
  2. Click "Test Connection" to verify
  3. Navigate to any web page (including logged-in pages)
  4. Click the Hermind extension icon in your browser toolbar
  5. Choose "Text" or "HTML" format
  6. Click "Send to Hermind"
  7. In Hermind, ask the AI about the page using the browser_extension_read tool
